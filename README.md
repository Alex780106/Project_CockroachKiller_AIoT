* 本戰車為一主動式滅蟑智慧戰車，主要功能如下：
1. 自動巡守家園並迴避障礙物
2. 自動追蹤並噴灑藥劑殺滅蟑螂
2. 回傳任務訊息給使用者的Line

* 硬體及小車環境建置
本戰車為使用JetBot AI智能小車改裝而成，搭載Jetson nano 2GB 主板\
